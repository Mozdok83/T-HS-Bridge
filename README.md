## О проекте

T-HS Bridge — портативное приложение (180 МБ), которое позволяет ИИ управлять вашим ПК из любого чата (DeepSeek, ChatGPT, Claude). Работает с флешки на любом компьютере — от Pentium I до современных ПК. Не требует установки.

## 🛡️ Система безопасности

- Цветовое кодирование опасности (🟢 безопасные / 🟡 опасные / 🔴 критические)
- Все опасные операции требуют подтверждения пользователя
- Семантический анализ PowerShell
- Критические команды с красными предупреждениями

## 🔌 Система плагинов

- Динамическая загрузка без перезапуска
- Включение/отключение на лету
- **CloneCore** — плагин памяти личности ИИ (решает проблему "амнезии")
- Есть  и другие плагины к примеру плагин который позволяет добавлять нехватающие команды
- ИИ пишет  плагины сам под нужды 

**Ключевые возможности:**
- 300+ команд (файлы, сеть, FTP, SSH, PowerShell, реестр)
- Графический интерфейс на PyQt6
